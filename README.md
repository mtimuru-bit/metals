# Металлы — Premium Collection 2026

Одностраничный HTML-документ для 3ДТЕМА (опт + розница): «Премиальные решения для интерьеров» — линейка из 6 авторских техник для мебельных фасадов МДФ. Полностью переработан под финальный PDF дизайнера (`_source/Premium_interior_2026_3dtema.pdf`, 2026-06-16).

## Назначение

- Обучение менеджеров и партнёров линейке покрытий.
- Можно отправлять клиентам (оптовым и розничным) как ознакомительный буклет.
- Внизу — отдельный раздел «Партнёрам: как продавать?» для менеджеров (Кому/Почему/Приёмы/Возражения).

## Структура документа

`index.html` — единственный файл презентации. Открывается в любом браузере, печатается в PDF (Ctrl+P → «Сохранить как PDF»).

Шапка: бейдж «Premium Collection 2026», заголовок «Премиальные решения для интерьеров». Нав-чипы: Жидкий, Водный, Вставки из шпона, Масла, Поталь CREPA, Вставки из металла, Фрезеровка под патину, Пилотные образцы, Условия, Партнёрам.

Разделы:

1. **Жидкий металл** — от 33 000 ₽/м², 4 цвета (золото, бронза, медь, олово). До 95% металлической пудры. Защитный лак/воск в цене. Блок «Доступные покрытия» — две степени подготовки: **полировка** (только гладкие/гладкие текстурные) и **шлифовка** (гладкие + 2D/3D). На фрезерованных шлифуется полуматово, в углублениях остаётся эффект патины. Наценка за фрезеровку и текстуру (3 базовых или индивидуальная).
2. **Водный металл** — от 19 800 ₽/м², **12 базовых оттенков** (в каталоге показаны 2 показательных — Старая бронза, Титан; полная палитра у менеджеров). Наценка за фрезеровку, 2D и 3D. Базовый вариант **без** брашировки, вставки шпона и лака. Опционально — брашировка, матовый лак, **вставки из шпона** (терминология обновлена 2026-06-16, было «инкрустация шпона»).
3. **Вставки из шпона** (подраздел Водного металла, отдельная ссылка в нав, якорь `#veneer`) — любой вид шпона с открытыми порами под матовым лаком, любая форма по эскизу (наценка за отрисовку), торцы морилкой, стоимость шпона по прайсу.
4. **Декоративные масла** — **13 оттенков Renner** (Tiziano, Tintoretto, Raffaello, Michelangelo, Masaccio, Giotto, Leonardo, Cimabue, Boticelli, Caravaggio, Mantegna, Donatello + 13-й) и **3 фактуры** (мятая ткань, москито, чешуя). Любая фактура × любой цвет. Базовые в каталоге — Leonardo / Boticelli / Cimabue. Цена уточняется. Веер палитры — `images/oils/oils-palette-renner.jpg` (извлечён из PDF дизайнера, стр.16).
5. **Поталь CREPA** (переименована с «Крепа») — листы 0,02 мм, золото = сплав меди и цинка, серебро = алюминий, медь = чистая медь. Стыки ~14×14 см. Защитный лак обязателен в цене. Реставрация переклейкой.
6. **Вставки из металла** (переименована с «Инкрустация металла») — нержавеющая сталь **1,2 мм** (сатин или полировка), готовые фигуры: Птица 1 (50×70 сатин), Птица 2 (50×70 полировка), Птица 3 (74×110 полировка), Заяц (50×70 сатин), Цветок 1 (70×50 полировка), Цветок 2 (70×58 полировка). На заказ — любая форма, наценка только за «Отрисовку». Установка после покраски/лакировки.
7. **Фрезеровка под патину** (отдельный раздел 06, вынесена из подраздела вставок) — рисунок фрезеруется в фасаде и патинируется. Готовый элемент «Стая» 300×400 мм в **двух цветах: золото и серебро**. Фасады в эмали, шпоне, **плёнке ПВХ**. Патина любого цвета, лак матовый/глянцевый.

Фактоиды цен везде по новой схеме PDF: «Цена от … на гладких фасадах без фрезеровки», «Наценка за фрезеровку, 2D и 3D фасады», «Образец ≈7–10 дней (300×400 / 200×200)», «Минимум 1 м²».

**Блок «Условия изготовления пилотных образцов»** (новый, 2026-06-16): пилотник = 0,5 м² готового изделия, скидка 25% на образец согласованного цвета, неограниченное число пилотников. Итог: 50% + 25% возвращается на утверждённый цвет, при размещении партии — ещё 25% (в сумме 75% компенсации).

**Раздел «Партнёрам: как продавать?»** (новый, 2026-06-16, тёмный фон, лаймовые акценты): Кому продавать (дизайнеры, ремонт под ключ, эксклюзив, коммерция), Почему покупают (статус, живая поверхность, настоящий металл, защита/реставрация), Сигналы «горячего» клиента, Приёмы продаж, Ответы на возражения, Как формируется цена и следующий шаг.

Общий блок в конце: срок образца ~7–10 дней, заказ от 1 м², обязательное согласование по образцу, специфика ручной работы.

## Хостинг и деплой

**Текущий прод (временный, для коллег): GitHub Pages** → `https://mtimuru-bit.github.io/metals/`. Репозиторий `mtimuru-bit/metals`, ветка `main`, source `/ (root)`.

Обновление через веб: репозиторий → **Add file → Upload files** → перетащить `index.html` + папку `images` (содержимое, не саму папку metals-guide) → **Commit changes**. Pages пересобирается за 1–2 минуты. Старые файлы веб-загрузка не удаляет (висяки безвредны).

**Постоянный дом — база знаний в Митинге** (`mm.3dtema.pro`, Yandex Cloud). Тимур загружает презентацию туда сам (можно бандлом). См. память `project-mityng-education`.

### Урок по хостингу (важно)

- **Netlify (`*.netlify.app`) и Cloudflare (`*.workers.dev`) НЕ работают из РФ** — throttle / `ERR_CONNECTION_RESET` (режет РКН). Не использовать для РФ-аудитории. Работают: GitHub Pages, Google (Apps Script), Yandex Cloud.
- **Картинки сжаты под веб** (≤1600px, прогрессивный JPEG ~Q82): исходные iPhone-фото по 1,5–2,5 МБ грузились слишком долго. Итог ~14 МБ против 42.
- **Если паковать в zip** (для базы знаний) — НЕ использовать PowerShell `Compress-Archive` (кладёт пути с `\`, Linux-распаковщик теряет папки). Только Python:

  ```bash
  python -c "import os, zipfile; z=zipfile.ZipFile('metals-guide.zip','w',zipfile.ZIP_DEFLATED); [z.write(os.path.join(r,f), os.path.relpath(os.path.join(r,f),'metals-guide').replace(os.sep,'/')) for r,d,fs in os.walk('metals-guide') if (d.__setitem__(slice(None),[x for x in d if x!='_source']) or True) for f in fs]; z.close()"
  ```

  (исключает `_source/`, чтобы PDF-исходник не попал на хостинг)

**Почему zip, а не папка:** при drag-drop папки уже один раз словили частичную заливку — Netlify зацепил половину `index.html` (21KB из 57KB). Zip атомарен — либо доходит целым, либо не доходит вовсе.

## Как обновлять

Источник правды — этот HTML-файл. Цены, тексты, добавление новых цветов / фактур / фото — редактируй прямо в `index.html`, структура карточек повторяющаяся:

- цветовая карточка: блок `.color-card` с `.color-swatch` (туда `<img>` или CSS-градиент) и `.color-info` (название + описание)
- фактурная карточка: блок `.texture-card` с `.texture-img` и `.texture-info`
- фактоид: блок `.fact` внутри `.facts` — «label» сверху, «value» снизу с опциональным `.unit`
- предупреждение / акцент: блок `.note` (мягкий) или `.care-panel` (с боковой полосой)
- логотип: блок `.logo-bar` (тёмная полоса сверху с белым логотипом) и `.footer-brand img` в подвале — оба ссылаются на `images/logo/logo-pro.png`. Если логотип на сайте 3dtema.pro поменяется, перекачай файл по тому же пути.
